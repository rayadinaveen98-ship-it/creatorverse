# CreatorVerse V1 Demo Specification

Build a premium web application called CreatorVerse.

## Purpose

CreatorVerse is a visual storytelling platform where users explore the journeys of great creators and understand how they became who they are.

The experience should feel cinematic, immersive, and editorial.

Think premium streaming service UI, not Wikipedia or IMDb.

---

# Design Style

* Dark theme
* Elegant typography
* Premium feel
* Mobile first
* Responsive
* Smooth hover animations
* Large visual cards
* Modern spacing
* Gold accent color
* Black background

Colors:

Background: #0B0B0B

Primary Text: #FFFFFF

Secondary Text: #AAAAAA

Accent: #D4A24C

# Technical Requirements

Use:

* Next.js
* TypeScript
* Tailwind CSS

Use placeholder images where necessary.

Focus on appearance and experience.

Do not build authentication.

Do not build backend.

Do not build database.

Use hardcoded data.

Goal:

Create a realistic demo that can be shown to potential users and gather feedback.
